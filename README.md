# SEN SEL (센셀) - 교실 속 보이지 않는 마음을 읽는 AI 사회 정서 레이더

학생들의 정서 건강과 교우 관계를 실시간으로 파악하고, 교사에게 맞춤형 지도 계획과 자리 배치까지 제안하는 웹 서비스 'SEN SEL'을 제출합니다.

---

## 1. 기획 의도

교실 현장에서 교사는 수십 명의 학생들을 동시에 관리합니다. 겉으로 밝아 보이는 아이가 속으로는 고립감을 느끼거나, 교우 관계에서 남모를 어려움을 겪고 있을 때 이를 조기에 발견하기란 쉽지 않습니다. 기존의 종이 설문조사는 형식적인 답변이 나오기 쉽습니다.

그래서 사춘기에 들어서는 초등학교 5학년을 지도하고 있는 담임 교사인 저는 "아이들이 가장 편하게 속마음을 털어놓을 수 있는 'AI 챗봇'을 활용해 학급의 정서 데이터를 자연스럽게 수집하면 어떨까?"라는 아이디어에서 출발했습니다. '짝꿍이 되고 싶은 친구는 누구야?', '같은 모둠이 되고 싶은 친구는 누구야?' 등과 같이 도덕과 동료평가법 중 추인법 기법을 활용해 AI 챗봇이 학생에게 대화를 건네고, 이를 토대로 학생들의 자신·타인과의 관계를 파악하고 지원할 수 있도록 구상했습니다.

나아가 수집된 데이터가 '보는 것'에서 끝나지 않고 **교실의 실제 의사결정(자리 배치, 모둠 구성, 위기 개입)으로 이어지도록** 기능을 고도화했습니다.

## 2. 핵심 기능

### 1) 자연스러운 학생 마음 데이터 수집 (학생용 AI 챗봇)

- 학생들은 귀여운 나무 캐릭터 챗봇과 일상적인 대화를 나눕니다. 프로필 생성 시 실명·닉네임·성별·오늘의 기분을 입력하며, 이전에 쓰던 실명을 입력하면 데이터가 그대로 복원됩니다.
- AI 챗봇(Gemini API 활용)이 대화의 맥락을 분석하여 학생의 현재 기분(건강/보통/힘듦)과 교우 관계(긍정적 지목) 데이터를 자동으로 추출합니다.
- **갈등·고립 신호 감지(고도화)**: 챗봇은 부정적 질문("싫은 친구는 누구야?")을 절대 던지지 않는 원칙을 유지하면서, 마음을 여는 개방형 질문("요즘 학교에서 마음이 불편했던 순간이 있었어?")을 자연스럽게 섞습니다. 학생이 **스스로** 친구와의 다툼·서운함·따돌림을 이야기하면 갈등 신호로, 외로움·혼자라는 표현을 하면 외로움 신호로 기록됩니다.
- '퍼지 매칭' 알고리즘을 적용하여 아이들이 친구 이름을 "정민이", "정민", "정민이랑" 등으로 다양하게 입력해도 정확하게 한 명의 학생으로 매핑합니다.

### 2) 실시간 학급 관계망 & 감정 시각화 (교사용 대시보드)

- 학생들의 데이터는 즉각적으로 교사용 대시보드에 연동됩니다.
- 별자리 테마의 소시오그램(Sociogram): 긍정적 지목을 주고받은 학생들 사이에 빛나는 화살표가 실시간으로 연결되며, 인기가 많은 학생일수록 별의 크기가 커집니다.
- 학급 감정 신호등: 학급 전체의 감정 상태를 통계 및 직관적인 파이 차트로 한눈에 파악할 수 있습니다.

### 3) 관계 신호 모니터링 (고도화 신규)

- **갈등 신호**: 학생이 자발적으로 언급한 갈등 관계를 목록으로 보여주며, 양쪽 모두 서로를 언급한 '상호 갈등'은 우선 개입 대상으로 강조됩니다.
- **고립 위험(연결 고리 없음)**: 긍정 지목을 주지도 받지도 못해 관계망에서 끊긴 학생을 자동 감지합니다.
- **외로움 신호**: 챗봇 대화에서 외로움을 표현한 학생을 횟수·최근 발생일과 함께 표시합니다.
- 이 신호들은 아래 자리 배치 AI에 자동으로 반영됩니다.

### 4) 학생 명단 관리 & 드래그 앤 드롭 자리 배치도 (고도화 신규)

- **학생 관리**: 교사가 학생을 개별 추가하거나 명단을 일괄 등록(`실명,닉네임,성별` 형식)할 수 있고, 성별(남/여)은 명단에서 클릭 한 번으로 지정·변경됩니다. 미리 등록된 실명으로 학생이 입장하면 데이터가 자동 연결됩니다.
- **자리 배치도**: 학생을 좌석으로 드래그하여 배치하고, 좌석끼리 드래그하면 자리가 교환됩니다. 줄 수·자리 수 조절, 자동 배치, 랜덤 섞기를 지원하며 배치는 Firestore에 저장되어 유지됩니다. 이름 색(파랑=남, 분홍=여)과 좌석 테두리 색(기분 상태)으로 학급 상황이 한눈에 보입니다.
- **모둠 지정**: 좌석을 클릭으로 선택해 모둠으로 묶으면 같은 모둠이 같은 색 점선 테두리로 표시됩니다. 모둠은 좌석에 귀속되어 학생을 바꿔 앉혀도 모둠 구역이 유지됩니다.
- **인쇄**: 학급명·날짜·범례가 포함된 인쇄 전용 레이아웃으로 자리 배치도를 바로 출력할 수 있습니다.
- **갈등 인접 경고**: 갈등 신호가 있는 두 학생이 인접 좌석에 배치되면 화면에 실시간 경고가 표시됩니다.

### 5) AI 자리 배치 제안 (고도화 신규)

- 학생들의 기분 상태, 긍정 지목 관계, 갈등 신호, 외로움 신호, 성별 데이터를 종합 분석하여 AI가 자리 배치안을 제안합니다.
- 배치 원칙: ① 갈등 신호가 있는 학생은 인접 배치 금지·분리 ② '힘듦' 학생은 교사 관찰이 쉬운 앞줄에 분산 ③ 고립·외로움 학생은 관계가 활발하고 건강한 학생 옆에 ④ 상호 지목 쌍은 인접 배치 ⑤ 기분·성별 고른 분산.
- 제안 모달에서 **전체 배치 근거(3~5문장)와 주요 학생별 배치 이유**를 함께 보여주고, 실명이 표시된 미리보기 배치도로 확인 후 적용합니다. 적용 후에도 드래그로 자유롭게 수정할 수 있습니다.

### 6) 위기 학생 AI 맞춤 처방 및 리포트 자동화

- 기분이 '힘듦' 상태인 학생(고립 위험군)을 자동으로 분류하여 대시보드에 띄워줍니다.
- 선생님이 미리 설정해 둔 SEL(사회정서학습) 가이드라인을 바탕으로, AI가 해당 학생을 어떻게 지도하면 좋을지 동료 베테랑 교사처럼 구체적인 맞춤 처방을 제공합니다.
- 추출된 처방은 '학급 정서 종합 리포트'에 자동으로 연동되며, 교사가 직접 텍스트를 수정·보완하여 곧바로 PDF로 인쇄해 학급 일지로 활용할 수 있습니다.

## 3. 강력한 안전 및 프라이버시 보호

- **최고 수준의 AI 안전 필터**(BLOCK_LOW_AND_ABOVE): 증오, 폭력, 성적, 위험 콘텐츠를 원천 차단하여 초등학생에게 안전한 환경을 제공합니다.
- **초등학생 맞춤형 대화 가이드라인 강제**: 정치, 유해 매체, 성인용 주제가 나오면 챗봇이 단호히 거절하고 학교 생활과 관련된 주제로 화제를 전환하도록 시스템 프롬프트를 고도화했습니다. 갈등 이야기를 들어도 잘잘못을 판단하지 않고 공감하며, 필요 시 교사와의 대화를 권합니다.
- **API 키 보안**: 모든 API 키는 코드에 하드코딩하지 않고 환경 변수(.env)로 분리했으며, Gemini 호출은 Vercel Serverless Function을 거쳐 키가 브라우저에 절대 노출되지 않습니다. (설정 방법: `ENV_SETUP_GUIDE.md` 참고)
- **개인정보 익명화 분석**: AI 자리 배치 분석 시 학생의 실명·닉네임을 외부로 전송하지 않습니다. 브라우저에서 익명 ID(S1, S2...)로 변환한 기분·관계 데이터만 전송하고, 분석 결과 속 익명 ID를 교사 화면에서만 실명으로 복원합니다.

## 4. 기술 스택 (Tech Stack)

- **Frontend**: React (Vite), react-force-graph-2d (관계망 시각화), Recharts, HTML5 Drag and Drop API (자리 배치)
- **Backend & Database**: Firebase (Firestore 실시간 DB, Authentication), Vercel Serverless Functions
- **AI Integration**: Google Gemini API (대화 분석, 맞춤 처방, 자리 배치 제안)

## 5. 시작하기

```bash
npm install                      # 의존성 설치
copy .env.example .env.local     # 환경 변수 파일 생성 후 실제 값 입력
vercel dev                       # 로컬 실행 (/api 서버리스 함수 포함)
```

환경 변수 설정과 Vercel 배포 방법은 [ENV_SETUP_GUIDE.md](./ENV_SETUP_GUIDE.md)에 자세히 정리되어 있습니다.

## 6. 기대 효과

SEN SEL은 교사의 업무 부담을 획기적으로 줄여주는 동시에, 학급 내 정서적 사각지대를 없애줍니다. 챗봇이 수집한 마음 데이터가 관계망 시각화를 넘어 자리 배치·모둠 구성이라는 교실의 실제 의사결정까지 이어지고, 갈등과 고립의 신호를 조기에 포착해 데이터 기반의 따뜻하고 정확한 개입을 가능하게 합니다. '모든 아이들이 심리적으로 안전함을 느끼는 교실'을 만들어가는 것이 최종 목표입니다.

감사합니다.
