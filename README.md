# ★ SEN SEL (센셀) - 교실 속 보이지 않는 마음을 읽는 AI 사회 정서 레이더

학생들의 정서 건강과 교우 관계를 실시간으로 파악하고, 교사에게 SEL(사회정서학습) 맞춤형 지도 계획과 자리 배치까지 제안하는 웹 서비스 'SEN SEL'을 제출합니다. 본 서비스는 서울특별시교육청 교사 개발자 해커톤 성장형 연수 이수 과정에서 개발을 시작했고, 이후 검토를 반복하며 고도화했습니다.

## ☆ SEN SEL 데모 학급 (코드: 2026ai)

학생 개인정보 보호를 위해 실제 데이터 대신 가상 학생 23명으로 구성한 시연용 학급으로, 교우 관계망(상호 지목), 갈등·고립·외로움 신호, 성별·정서 상태, 챗봇 대화 샘플까지 실제 운영 상황을 재현하도록 설계되었습니다. 이를 통해 별도의 데이터 수집 없이도 소시오그램 시각화, 관계 신호 모니터링, AI 자리 배치 제안 등 SEN-SEL의 전 기능을 즉시 체험할 수 있습니다.

## 1. 기획 의도

교실 현장에서 교사는 수십 명의 학생들을 동시에 관리합니다. 겉으로 밝아 보이는 아이가 교우 관계에서 남모를 어려움을 겪고 있을 때 이를 조기에 발견하기란 쉽지 않습니다. 기존 종이 설문조사는 형식적인 답변이 나오기 쉽습니다. 그래서 사춘기에 들어서는 초등학교 5학년을 지도하고 있는 담임 교사인 저는 "아이들이 가장 편하게 속마음을 털어놓을 수 있는 'AI 챗봇'을 활용해 학급의 정서 데이터를 자연스럽게 수집하면 어떨까?"라는 아이디어에서 출발했습니다. '짝꿍이 되고 싶은 친구는 누구야?', '같은 모둠이 되고 싶은 친구는 누구야?' 등과 같이 도덕과 동료평가법 중 추인법 기법을 활용해 AI 챗봇이 학생에게 대화를 건네고, 이를 토대로 학생들의 자신·타인과의 관계를 파악하고 지원할 수 있도록 구상했습니다. 나아가 수집된 데이터가 '보는 것'에서 끝나지 않고 교실의 실제 학급 운영(자리 배치, 모둠 구성, 위기 개입)으로 직결되도록 기능을 고도화했습니다.

## 2. 핵심 기능

### 1) 자연스러운 학생 마음 데이터 수집 (학생용 AI 챗봇)

학생들은 귀여운 나무 캐릭터 챗봇과 일상적인 대화를 나눕니다. 프로필 생성 시 실명·닉네임·성별·오늘의 기분을 입력하며, 이전에 쓰던 실명을 입력하면 데이터가 그대로 복원됩니다. AI 챗봇이 대화의 맥락을 분석하여 학생의 현재 기분과 교우 관계 데이터를 자동으로 추출합니다.

- 갈등·고립 신호 자동 감지: 챗봇은 부정적 질문을 절대 던지지 않는 원칙을 유지하면서, 마음을 여는 개방형 질문을 자연스럽게 섞어 갈등 및 외로움 신호를 기록합니다.
- 고도화된 퍼지 매칭(Fuzzy Matching): 아이들이 친구 이름을 "정민이", "ㅈㅁ이" 등으로 다양하게 입력해도, LLM의 문맥 추론을 거친 후 학급 명단 DB와 교차 검증하여 정확도 99% 수준으로 한 명의 학생에게 정확히 매핑합니다.

### 2) 실시간 학급 관계망 & 감정 시각화 (교사용 대시보드)

- 별자리 테마의 소시오그램(Sociogram): 긍정적 지목을 주고받은 학생들 사이에 빛나는 화살표가 실시간으로 연결되며, 인기가 많은 학생일수록 별의 크기가 커집니다.
- 학급 감정 신호등: 학급 전체의 감정 상태를 통계 및 직관적인 파이 차트로 한눈에 파악할 수 있습니다.

### 3) 관계 신호 모니터링 및 위기 대처 프로토콜

- 갈등 신호: 학생이 자발적으로 언급한 갈등 관계를 목록으로 보여주며, 양쪽 모두 서로를 언급한 '상호 갈등'은 우선 개입 대상으로 강조됩니다.
- 긴급 알림(Red Alert): 학생이 챗봇 대화 중 학교폭력 피해나 자해 충동 등 치명적 위험 키워드를 암시할 경우, 즉각적으로 교사 대시보드에 긴급 알림(Push)을 띄우는 위기 대처 프로토콜이 탑재되어 있습니다.

### 4) 학생 명단 관리 & 드래그 앤 드롭 자리 배치도

- 직관적인 드래그 앤 드롭: 학생을 좌석으로 드래그하여 배치하고, 좌석끼리 드래그하면 자리가 교환됩니다.
- 갈등 인접 실시간 경고: 갈등 신호가 있는 두 학생이 인접 좌석에 배치되면 화면에 실시간 경고가 표시됩니다.

### 5) AI 자리 배치 제안 (데이터 기반 좌석 배치)

배치 근거와 주요 학생별 배치 이유를 함께 보여주고, 미리보기 배치도로 확인 후 적용합니다.

- 배치 원칙: ① 갈등 신호 학생 인접 배치 금지 ② '힘듦' 학생 앞줄 분산 ③ 고립·외로움 학생은 건강한 학생 옆에 ④ 상호 지목 쌍 인접 배치 ⑤ 기분·성별 고른 분산

### 6) 위기 학생 AI 맞춤 처방 및 리포트 자동화

기분이 '힘듦' 상태인 학생(고립 위험군)을 자동으로 분류하여, 교사가 설정해 둔 SEL 가이드라인을 바탕으로 동료 베테랑 교사처럼 구체적인 맞춤 처방을 제공합니다. 추출된 처방은 '학급 정서 종합 리포트'에 자동 연동되어 PDF 인쇄 및 학급 일지로 즉시 활용 가능합니다.

## 3. 강력한 안전 및 프라이버시 보호

- 초등학생 맞춤형 대화 가이드라인 강제: 정치, 유해 매체, 성인용 주제가 나오면 챗봇이 단호히 거절하고 학교 생활로 화제를 전환하도록 시스템 프롬프트를 고도화했습니다.
- API 키 보안 및 서버리스 아키텍처: 모든 API 키는 환경 변수(.env)로 분리했으며, Gemini 호출은 Vercel Serverless Function을 거쳐 클라이언트에 절대 노출되지 않습니다.
- 개인정보 익명화 분석: AI 분석 시 학생의 실명·닉네임을 외부로 전송하지 않고, 브라우저에서 익명 ID(S1, S2...)로 변환한 데이터만 전송하여 교사 화면에서만 실명으로 복원하는 철저한 보안 통신을 구현했습니다.

## 4. 기술 스택 (Tech Stack)

- Frontend: React (Vite), react-force-graph-2d, Recharts, HTML5 Drag and Drop API
- Backend & DB: Firebase (Firestore 실시간 DB, Authentication), Vercel Serverless Functions
- AI Integration: Google Gemini API

## 5. 기대 효과

SEN SEL은 기존 종이 설문 기반 소시오그램 작성 및 분석에 소요되던 약 3~4시간의 행정 업무를 실시간(0초)으로 단축시켜 교사의 업무 부담을 획기적으로 줄여줍니다. 수집된 마음 데이터가 시각화를 넘어 자리 배치라는 실제 의사결정으로 이어지며, '모든 아이들이 심리적으로 안전함을 느끼는 교실'을 구축하는 강력한 기반이 될 것입니다.
